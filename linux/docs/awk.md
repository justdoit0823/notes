

使用
========

内部变量
--------

  * NF


		存储单行记录中有多少字段。

  * NR

		存储记录条目数。

  * $0

		单行内容。

  * $i

		单行中第i个字段的内容。


命令语法
--------

	BEGIN match_apttern commands END

	支持if条件语句，for语句。


字符串操作函数
--------------

  * substr

  * split

  * length
